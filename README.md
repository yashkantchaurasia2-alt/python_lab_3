# Library Inventory Manager

A simple command-line Library Inventory Manager for managing books in a campus library.  
It demonstrates object-oriented design, JSON file handling, exception handling, and logging.

## Features

- Add books with title, author, and ISBN.
- Issue and return books, with status tracking (available / issued).
- Search books by title or ISBN.
- View all books in the catalog.
- Persistent storage of books in a JSON file.
